# 💡 LUMA 2.0 - W Command Center

O **LUMA 2.0** é uma extensão de alta performance para Google Chrome, desenvolvida para transformar a experiência de suporte e análise de dados dentro do ecossistema Clinicorp. Focada em **hiper-produtividade**, a ferramenta injeta uma interface inteligente (UI) e motores de automação diretamente no navegador.

## 🚀 Funcionalidades Principais

* **TagSense (T):** Sistema de categorização inteligente que utiliza IA para analisar conversas e sugerir as tags mais precisas de Área e Tipo.
* **Varredura (V):** Motor de busca e análise de protocolos em lote, permitindo identificar padrões e resolver chamados com velocidade recorde.
* **W Interface:** Um Floating Action Button (FAB) totalmente customizado com estados de "sono" e interações fluidas para não atrapalhar o fluxo de trabalho.
* **Exportação de Dados:** Integração com `ExcelJS` para gerar relatórios detalhados diretamente da interface da extensão.

## 🛠️ Stack Técnica

* **Core:** JavaScript Moderno (ES6+)
* **Arquitetura:** Chrome Extension Manifest V3
* **UI/UX:** CSS3 com variáveis dinâmicas e efeitos de blur (Glassmorphism)
* **Integrações:** Gemini AI (via API), ExcelJS para manipulação de planilhas.
